# RESTful 작업
------
 **RESTful 웹 형식**은 웹에서 클라이언트(프론트엔드)와 서버(백엔드)가 **HTTP를 이용해 데이터를 주고받는 방식 중 하나.** 이 형식을 통해 벡엔드와 프론트 엔드가 구분됨.
